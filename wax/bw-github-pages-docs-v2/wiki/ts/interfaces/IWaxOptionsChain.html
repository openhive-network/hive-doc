OptionalwasmLocation

wasmLocation?: string

The path to the WASM file. It can be a relative path or an absolute URL If not specified, the default path is used: "./build_wasm/wax.common.wasm" (may change if bundled)

Note: You can also specify a base64 encoded string of the WASM file to be used directly when inlining

Note: When you don't have your WASM file served from the static directory (such as /public), or you are bundling / transpiling your code, you may need to leverage your bundler's asset importing capabilities, e.g.:

     - For Vite:    `import wasmUrl from './my_wasm_files/wax.common.wasm?url';`
     - For Webpack: `const wasmUrl = new URL("./my_wasm_files/wax.common.wasm", import.meta.url).href;`
  Then pass the imported `wasmUrl` variable to this option.
  For web workers, replace `import.meta.url` with `self.location.href`.
